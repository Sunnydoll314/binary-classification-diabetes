# Project Title: Diabetes Prediction

## 1. Introduction
This project aims to predict whether a patient has diabetes using machine learning models.

## 2. Dataset
- Source: Pima Indians Diabetes Dataset
- Features include: Glucose, BMI, Age, etc.

## 3. Data Cleaning
- Replaced invalid zeros with NaN
- Imputed missing values using median

## 4. Exploratory Data Analysis (EDA)
- Used bar charts and histograms
- Observed relationships between features and outcome

## 5. Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## 6. Evaluation
- Accuracy
- Confusion Matrix
- Precision & Recall

## 7. Results
- Random Forest performed the best

## 8. Conclusion
- Certain features like Glucose are strong predictors
